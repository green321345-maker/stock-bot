# Stock Bot Pro KR

한국/미국 주식을 함께 지원하는 **실행 가능한** 투자 리서치 앱입니다.

## 프리미엄급 기능

- 한국주식 + 미국주식 검색 (`005930` 같은 6자리 코드도 지원)
- 장기 차트: `6mo`, `1y`, `3y`, `5y`, `10y`, `max`
- 원클릭 우량기업 스크리너 (전략점수 + 신뢰도 기준)
- 시가총액 필터(대형/중형/소형) + 필터 검증 요약
- 전략 프리셋(복수 선택): Buffett / Lynch / Graham / CANSLIM Lite / Minervini Lite / Quality Growth
- 통과 전략/미통과 전략 표시 + 제외 사유 상세
- 통계 한글화 탭 (PER, PBR, ROE, 부채비율, 성장률 등)
- 리스크 통계(기간 수익률, 연환산 변동성, MDD)
- 전문가 의견/목표가 + 출처 안내
- 신뢰도 기반 예상 주가 (가중치 공개)
- 관심종목 저장 + Webhook 기반 모바일 알림(Discord 지원)
- 자동 감시 모드(준실시간): 일정 주기 자동 점검/전송 + 중복 쿨다운
- 완전 자동 백그라운드 알림 워커(`auto_alert_worker.py`)
- 종목별 알림 조건 저장(`alert_rules.json`)
- 뉴스 탭
- 투자 스타일(스캘핑/스윙/장기) + 투자금 구간별 조언
- 이슈 기반 추천/주의 섹터 가이드(금리/인플레/침체/AI 사이클)
- 스타일/투자금 적합성 필터 + 제외 사유 제공
- 리스크 기반 포지션 사이징 계산기(계좌 통화(KRW/USD) + 계좌금액/허용손실 기반)

## 디자인

- 전체 UI를 가독성 높은 밝은 테마로 구성
- 귀여운 파스텔 배경(핑크+보라 그라데이션 포인트) 적용
- 은은한 파스텔 보라 포인트 컬러 사용 (`#B79CFF`, `#6E4CCB`)

## 실행

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./run.sh
```


엔트리 파일: `stock.py`


## 알림 받는 방법

현재 앱의 알림은 **Webhook 방식**입니다.

1. 알림 받을 채널을 먼저 준비
   - Discord Webhook
   - Slack Incoming Webhook
   - Telegram 봇 + webhook 연결 서비스
   - IFTTT Webhooks + 스마트폰 앱 알림
2. 앱의 `관심종목/알림` 탭에서 종목별로 Webhook URL/매수/매도 기준 입력
3. `이 종목 알림 조건 저장` 버튼으로 저장
4. `조건 확인 후 알림 보내기` 버튼 실행
5. 조건이 맞으면 해당 채널로 즉시 메시지 전송
6. 자동 감시 모드를 켜면 페이지가 열려 있는 동안 주기적으로 조건을 점검합니다.

즉, 이 앱은 자체 푸시 서버 대신 **외부 알림 채널**과 연동해 폰 알림을 받는 구조입니다.


## 완전 자동 알림 실행

앱 화면을 켜두지 않아도 알림을 보내려면 워커를 실행하세요.

```bash
./run_worker.sh
```

기본값:
- 점검 주기: 60초
- 동일 신호 쿨다운: 10분

커스텀 실행:
```bash
python auto_alert_worker.py --interval 30 --cooldown 5
```
