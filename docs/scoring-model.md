# 점수 모델 설계 (Buffett + Lynch + 검증된 보조 전략)

## 1) 공통 원칙

- 0~100 정규화
- 결측치 많은 항목은 감점보다 신뢰도 하향
- 점수와 신뢰도를 분리 표기

## 2) Buffett Score (예시 가중치)

- 내재가치 대비 할인율: 30%
- 자본효율(ROIC/ROE): 20%
- 재무건전성(부채, 이자보상): 20%
- 수익 예측 가능성(마진/FCF 안정성): 20%
- 해자 프록시(점유율/브랜드/네트워크): 10%

## 3) Lynch Score (예시 가중치)

- PEG(성장 대비 가격): 35%
- 이익/매출 성장 지속성: 25%
- 이해가능성(산업 단순성/비즈니스 모델): 15%
- 재무건전성: 15%
- 밸류에이션 과열 여부: 10%

## 4) Composite Score

`Composite = w1*Buffett + w2*Lynch + w3*Quality - w4*Risk`

- 기본값: w1=0.35, w2=0.25, w3=0.25, w4=0.15
- 사용자 성향별 프리셋 제공

## 5) Confidence Score (신뢰도)

- 데이터 커버리지(40)
- 최신성(20)
- 소스 일치도(20)
- 모델 안정성/변동성(20)

등급:
- A: 85+
- B: 70~84
- C: 55~69
- D: <55

## 6) 검증

- 롤링 윈도우 백테스트 (최소 10년)
- 거래비용/슬리피지 반영
- 체리피킹 방지: 사전 정의된 유니버스 고정
